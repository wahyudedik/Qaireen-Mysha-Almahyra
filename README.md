# Love Forever Animation

This project creates a heart-shaped animation using HTML, CSS, and JavaScript. The animation displays the repeated text "Qaireen Mysha" forming a glowing heart shape.

## How it works
- The animation is rendered in the browser using dynamically generated HTML elements.
- CSS is used for styling and animation effects.
- JavaScript creates and arranges the elements to form the heart shape.

## Usage
1. Open `index.html` in your browser.
2. You will see the animated heart with the text "Qaireen Mysha".
